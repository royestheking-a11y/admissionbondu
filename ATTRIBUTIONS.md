- **Product**: Admission Bondhu — University Admission Support Platform in Bangladesh

## Key libraries used

- **React + Vite**: frontend app and build tooling
- **React Router**: client-side routing
- **MongoDB (Mongoose)**: database and models
- **Cloudinary**: media uploads (logos, PDFs) using an unsigned preset
- **Google Identity Services**: “Continue with Google” login/signup
